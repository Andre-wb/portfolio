02

Какие данные собираются

Сайт не требует регистрации и не хранит персональные данные в базе данных. Данные могут быть получены только в следующих случаях:

  • При заполнении формы составления технического задания — вы добровольно указываете описание проекта, бюджет и контактные данные (Telegram, email или телефон).
  • При отправке ТЗ через Telegram-бота — текст и прикреплённые файлы поступают напрямую мне в личные сообщения.
  • При прямом обращении через мессенджер или email — в рамках деловой переписки.

Сайт не использует счётчики аналитики и не передаёт данные рекламным системам.

03

Цели обработки данных

Информация, которую вы предоставляете при заполнении ТЗ, используется исключительно для:

  • Изучения требований к проекту и подготовки коммерческого предложения.
  • Связи с вами для уточнения деталей задачи.
  • Выполнения работ по договорённости.

Данные не используются в маркетинговых целях, не продаются и не передаются третьим лицам без вашего согласия.

04

Файлы cookie

Сайт использует файлы cookie — небольшие текстовые файлы, которые сохраняются в вашем браузере. Они не содержат личных данных.

На сайте используются следующие типы cookie:

  • Функциональные — сохранение прогресса при заполнении формы ТЗ (localStorage), чтобы вы не потеряли введённые данные при случайном закрытии страницы.
  • Согласие — запись вашего согласия с данной политикой, чтобы баннер не показывался повторно.

Сайт не использует рекламные, аналитические или отслеживающие cookie от третьих сторон.

Вы можете отключить cookie в настройках браузера в любой момент. Это может повлиять на работу формы составления ТЗ — прогресс перестанет сохраняться.

05

Сервис составления технического задания

На сайте доступен интерактивный конструктор технического задания. При его использовании:

  • Ответы на вопросы временно сохраняются в localStorage вашего браузера — исключительно для удобства (чтобы не начинать заново при обновлении страницы).
  • Данные не отправляются на сервер автоматически — только при явном нажатии кнопки «Отправить».
  • Прикреплённые файлы (макеты, изображения) передаются напрямую через Telegram Bot API и не хранятся на сервере сайта.
  • Текст ТЗ и контактные данные поступают только мне и нигде не публикуются.
06

Передача данных третьим лицам

При отправке ТЗ через кнопку «Отправить через бота» данные проходят через Telegram Bot API (Telegram Messenger Inc.). На эти данные распространяется политика конфиденциальности Telegram.

Сайт размещён на хостинге. Сервер обрабатывает входящие запросы, но не сохраняет содержимое ТЗ или персональные данные в базе данных.

Никакие данные не передаются рекламным сетям, брокерам данных или иным третьим лицам.
